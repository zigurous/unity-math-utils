# Extension Methods

A large majority of the functionality provided by the Math Utils package comes from extension methods. All of the extension methods found throughout the project are focused around numbers, whether it be arithmetic operations, bitwise operations, math algorithms, and other utilities of varying complexity. Extensions are available for the following data structures and classes:

- [Array](xref:Zigurous.Math.ArrayExtensions)
- [Bounds](xref:Zigurous.Math.BoundsExtensions)
- [Collider](xref:Zigurous.Math.ColliderExtensions)
- [Double](xref:Zigurous.Math.DoubleExtensions)
- [Float](xref:Zigurous.Math.FloatExtensions)
- [IComparable](xref:Zigurous.Math.ComparableExtensions)
- [Int](xref:Zigurous.Math.IntExtensions)
- [List](xref:Zigurous.Math.ListExtensions)
- [Long](xref:Zigurous.Math.LongExtensions)
- [Quaternion](xref:Zigurous.Math.QuaternionExtensions)
- [Short](xref:Zigurous.Math.ShortExtensions)
- [UInt](xref:Zigurous.Math.UIntExtensions)
- [Vector2](xref:Zigurous.Math.Vector2Extensions)
- [Vector2Int](xref:Zigurous.Math.Vector2IntExtensions)
- [Vector3](xref:Zigurous.Math.Vector3Extensions)
- [Vector3Int](xref:Zigurous.Math.Vector3IntExtensions)
- [Vector4](xref:Zigurous.Math.Vector4Extensions)
